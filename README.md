# Exercism Solutions - Python & SQL (Até o momento!)

Este repositório contém minhas resoluções para os desafios propostos na plataforma [Exercism](https://exercism.org/), focando no aprimoramento de lógica de programação, estruturas de dados e manipulação de bancos de dados.

O objetivo principal é documentar minha evolução técnica e aplicar as melhores práticas de desenvolvimento em cada track, preparando-me para desafios de nível LeetCode e entrevistas técnicas.

---

## Trilhas Atuais

### Python
Exercícios focados em:
* **Estruturas de Dados:** Listas, dicionários, sets e tuplas.
* **Algoritmos:** Busca, ordenação e manipulação de fluxos complexos.
* **Clean Code:** Aplicação de PEP 8 e legibilidade.
* **Paradigma OO:** Classes, métodos e encapsulamento.

### SQL (SQLite)
Exercícios focados em:
* **DQL (Data Query Language):** Consultas complexas e otimização.
* **Relacionamentos:** Uso estratégico de JOINs e subqueries.
* **Transformação de Dados:** Funções de agregação e lógica condicional em queries.
